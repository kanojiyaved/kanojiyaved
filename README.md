 ## 🚀 Projects
 
 - **💳 Credit Card Fraud Detection**  
   Built a Random Forest model to detect fraudulent transactions from financial datasets.
 
 - **🚗 Car Recommendation System**  
   Developed a content-based recommendation system using Python, Scikit-learn, and Streamlit.
 
 - **🎬 Movie Genre Prediction**  
   Implemented NLP techniques using TF-IDF and machine learning classifiers to predict movie genres.
 
 - **🎵 Spotify Song Recommender**  
   Built a recommendation system using audio features and similarity metrics to suggest songs.
 
 - **🏎️ F1 2026 — Chinese Grand Prix Podium Predictor**  
   An AI-powered Formula 1 podium prediction tool for the 2026 Chinese Grand Prix. It uses a weighted scoring model built on real FastF1 telemetry and historical race data to rank all 20 drivers and predict the top 3 finishers.
